# Bad Apple assets

Bundled files for the `badapple` easter egg (served from `/assets/bad-apple/`).

| File | Description |
|------|-------------|
| `framesData.lz` | LZ-compressed ASCII frames @ 30 FPS |
| `bad_apple.mp3` | Synced audio |

## Download into this folder

```bash
npm run bad-apple:fetch
```

Source: [EmirXK/bad_apple](https://github.com/EmirXK/bad_apple) (MIT).
